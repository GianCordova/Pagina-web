# Propuesta de diseño - Pagina Principal Fundacion Kinal

## Cambios realizados

Estas son las 5 cosas que modifique en la pagina principal:

1. Se cambio el diseño del inicio de la pagina agregando un encabezado algo mas llamativo, con un color azul encima para que el texto se vea mejor y resalte mas.

2. Se agrego una barra de navegacion que se queda fija en la parte superior cuando el usuario baja por la pagina, asi haciendo que la navegacion dentro del sitio.

3. Se agrego un apartado con un tipo de tarjetas donde se muestra informacion relevante como contacto, ubicacion de Kinal etc. Haciendo que esta informacion sea mas visible y facil de encontrar.

4. Se actualizaron los colores a uno tonos mas fuertes como azul y naranja, poniendolos en botones y detalles para darle una apariencia mas moderna y juvenil.

5. Los programas educativos se movieron a tarjetas, asi permitiendo que la pagina se vea bien tanto en computadoras como en celulares.
